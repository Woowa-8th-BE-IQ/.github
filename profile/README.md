### 💡 Why 'I Question'?

> "**나는 질문한다, 고로 존재한다.**"

* **지능의 새로운 패러다임:** LLM 시대에 파편화된 지식은 어디에나 있습니다. 이제 중요한 것은 '**어떤 질문을 던져 본질을 꿰뚫는가**'이며, 그것이 곧 개발자의 지능(Intelligence)이라고 믿습니다.
* **호기심과 본질 탐구:** "왜 이 코드가 동작하는가?"를 넘어 "왜 이 구조여야만 하는가?"를 끊임없이 묻습니다. 
* **본질을 꿰뚫는 개발자:** 표면적인 구현에 그치지 않고, 기술의 뿌리와 원리를 이해하려는 집요함을 지향합니다.
